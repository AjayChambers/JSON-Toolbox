# JSON Toolbox Formatting & Styling

<!-- Explains what this document is, and what it does -->
This file describes how code should be formatted/styled throughout the the _JSON Toolbox_ project,
however; this document describes the formatting/styling standards to use throughout the entire project,
and therefore, it extends to meta formatting specifications beyond just styling the project's C++ source
code, for example: file extensions, test naming, copyright/licensing documentation, and more.
